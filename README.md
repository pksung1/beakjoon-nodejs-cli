# 백준 문제 풀이 CLI 도구

백준 온라인 저지 문제 풀이를 위한 CLI 도구입니다. 문제 번호를 입력하면 템플릿 파일을 자동으로 생성하고, 정답 코드를 클립보드에 복사할 수 있습니다.

## 설치

### 글로벌 설치 (권장)

npm을 통해 글로벌로 설치하면 어디서든 `bjc` 명령어를 사용할 수 있습니다:

```bash
npm install -g @pksung1/beakjoon-nodejs-cli
```

설치 후 바로 사용 가능합니다:

```bash
# 문제 풀이를 위한 js파일과 test 파일생성
bjc 1000
# --> 1000.js, 1000.test.js 생성

# 문제 copy, console.log를 제거하고 prettier를 적용하여 코드를 복사합니다.
# 제출을 위해 입력을 읽어들이는 코드를 상단에 포함시킵니다.
bjc 1000 --copy
```

## 문제 풀이 방법

1. bjc 명령어로 문항을 생성합니다
2. test.js 파일에 백준문제의 테스트케이스를 추가합니다
3. js 파일의 solution 함수를 구현합니다

solution 함수는 return값을 console.log로 출력합니다.
**결과값은 숫자 혹은 문자열을 반환해야합니다**

4. 문제를 모두 푼경우, 제출을 위해 --copy 로 문제를 클립보드에 복사합니다. 복사를 하는경우, 내부의 console.log 코드를 제거하고 prettier를 적용해 복사합니다.
5. control + v 로 코드를 제출합니다.


## 주요 기능

### 1. 템플릿 파일 생성
- 문제 번호와 생성 날짜가 자동으로 주석에 포함됩니다
- `solution` 함수 템플릿이 제공됩니다
- 테스트 케이스 템플릿이 함께 생성됩니다

### 2. 정답 코드 복사
- `console.log` 문을 자동으로 제거합니다
- Prettier를 사용하여 코드를 포맷팅합니다
- 백준 제출용 prefix 코드를 자동으로 추가합니다
